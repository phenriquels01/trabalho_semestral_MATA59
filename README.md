## Objetivo do Projeto

A Superintendência de Tecnologia da Informação está implementando sua infraestrutura de rede local e de interconexão entre departamentos.

O projeto consiste em:

- Planejar, configurar e testar uma rede corporativa composta por 4 departamentos.
- Utilizar o **Cisco Packet Tracer** como ambiente de simulação.
- Abranger desde o planejamento de endereçamento até a integração **VLAN** com todos os serviços de rede ativos.
